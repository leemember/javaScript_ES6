# 🌹 웹 게임을 만들며 배우는 JS

## 자바스크립트 기초

- 값 : 숫자, 문자("",'' 안에 문자열 작성), 불린(참/거짓)<br>
- 연산자 : 1 + 3 입력하면 / 4로 출력 (+ 덧셈, - 뺄셈, / 나눗셈, \* 곱셈, % 나머지)<br>
- 연산자 부등호 : <, >, <= 작거나 크다 , >= 크거나 같다, == 같다<br>
- 비교 연산자 : === 3번치면 더 **명확하게** 계산해준다. 그래서 비교할 때는 === 세번 사용하는 것이 좋다.

```
5 == '5' // true
5 === '5' // false
```

- != 다르다
