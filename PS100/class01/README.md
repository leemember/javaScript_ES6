### 📃 문제 2번 : 배열의 내장함수

- <pass>부분에 배열 내장함수를 이용하여 코드를 입력하고 다음과 같이 출력되게 하세요.

```
데이터
var arr = [200,100,300];
//pass
console.log(arr);

출력
[200,100,10000,300]
```

✨splice✨ : 배열 안에 새로운 원소를 **삽입하거나 삭제하거나 교체하는 역할**을 한다. 이 함수는 3가지 매개변수가 필요한데 첫 번째 매개변수는 변화를 할 원소에 첫번째 인덱스를 의미한다.

<code>arr.splice(0, 0, 0);</code>
첫 번째 인덱스 값이 0이라면 arr의 200을 가르키는 것이다. 1이라면 arr 함수 배열에서 두 번째 원소 100을 의미하는거고, 2라면 세 번째 원소 300을 의미하는 것이다.

두 번째에 들어 갈 매개변수는 제거 할 원소의 갯수이다. 첫 번째가 0이고 두 번째 매개변수가 1이면 200부터 1개의 원소를 제거한다는 뜻이다. (즉, 200을 제거한다는 의미)

세번째 매게변수에 의미는 기존에 있던 원소를 삭제하고 새롭게 삽입 할 값을 의미한다.

첫번째 인덱스부터 1개를 삭제하고 10000을 집어넣으면
arr.splice(0, 1, 10000) 으로 구현하면 된다.
그럼 콘솔창에는 [10000, 100, 300] 이 찍힌다. 근데 우리의 목표는 [200, 100, 10000, 300]
이 값으로 출력되게 하는 것이니 <code>arr.splice(2, 0, 10000);</code>
으로 해주면 된다. 두 번째 매개변수가 0이면 아무것도 삭제되지 않는다.

### 📃 문제 5번 : for문 계산

- 다음 코드의 출력 값으로 알맞은 것은?

```
var a = 10;
var b = 2;

for(var i=1; i<5; i+=2) {
  a+=i;
}

console.log(a+b);
```

[보기]

1. 10
2. 12
3. 14
4. <code>16</code>

### 👉 문제 결과 : 4번 (16)

<br>

위 처럼 보면 분석하기 어려우니 식을 해제를 해보겠다.

🔸 i+=2 이 코드는 i=i+2의 축약이다.

🔸 a+=i; 이 코드는 a=a+i의 축약형이다.

🔸 i 변수는 1이고 i 는 5보다 작다. (i<5) 5와 같아지는 순간 for문은 종료

🔸 i 값은 매번 돌 때마다 2씩 증가한다. (i+=2)

처음에는 i가 1인 상태이고, 두 번째 돌 때는 1(i)+2가 되니
i는 3이 되고 세 번째 돌 때는 i=3+2 되었으니 답은 5가 된다.
그럼 가운데 매개변수 조건문인 i<5는 성립이 되지 않아 for문이 종료된다. 그럼 **최종적으로 for문은 총 2번만 반복된다는 것을 알 수 있다.**

a+=i는 a=a+i이니

🔸 첫 번째 반복 되었을 때 a=10+1 ====> a = 11

🔸 두 번째 반복 되었을 때 a=11+3 ====> a = 14

**결국 a는 14가 되었다.** 그리고 b는 원래 변수에 담겨있던 값 2이니 콘솔에 찍혀 나오는 a+b **최종값은 16이 된다.**

```
[데이터]
var year = '2019';
```

각 변수들에 문자열이 들어있다. 그리고 원하는 출력대로 <code>2019/04/26 11:34:27</code> 해주려면 <code>concat</code>이라는 메서드를 사용하면 된다. concat 메서드의 기능은 문자와 문자를 이어주는 메서드이다. **concat은 concatenate 라는 동사에서 왔다. "무엇과 무엇을 잇다. 즉, 연결하다"** 라는 뜻을 지닌다.

```
var year = "2019";
var month = "04";
var day = "26";
var hour = "11";
var minute = "34";
var second = "27";

var result = year.concat(
  "/",
  month,
  "/",
  day,
  " ",
  hour,
  ":",
  minute,
  ":",
  second

console.log(result);
```

year 뒤에 변수들을 연결하기 위해서 year.concat 메서드를 사용해 () 괄호 안에 연결 할 변수들과 <code>," ",</code> 를 이용하여 특수기호를 넣으면 된다.

### 📃 문제 10번 : 별찍기

- 크리스마스 날, 은비는 친구들과 함께 파티를 하기로 했습니다. 그런데, 크리스마스 트리 사는 것을 깜빡하고 말았습니다. 온 가게를 돌아다녀 봤지만 크리스마스 트리는 모두 품절이었습니다. 하는 수 없이 은비는 프로그래밍으로 트리를 만들기로 합니다.

![](https://images.velog.io/images/leemember/post/0f260537-df68-461d-8f12-dba0fcfd81c4/%E1%84%89%E1%85%B3%E1%84%8F%E1%85%B3%E1%84%85%E1%85%B5%E1%86%AB%E1%84%89%E1%85%A3%E1%86%BA%202021-06-08%20%E1%84%8B%E1%85%A9%E1%84%92%E1%85%AE%2010.58.53.png)

<br>

### 👉 문제 결과

```
const level = 5;

for (let i = 1; i <= level; i++) {
  let tree = "";

// 💡공백 증가의 반복문
  for (let k = 1; k <= level - i; k++) {
    tree = tree + " ";
  }

// 💡별 증가의 반복문
  for (let j = 1; j <= i * 2 - 1; j++) {
    tree = tree + "*";
  }

  console.log(tree);
}
```

문제와 같은 트리를 만들기 위해 입력문은 5를 제공해주었다. 그래서 변수 level에 담긴 값은 5로 선언했다. 그리고 전체적인 for문을 만들어주는데, 변수 i의 초기값을 1로 지정하고 i는 5인 level보다 작거나 같아야한다. 왜냐면 트리를 만들려면 5가 멕시멈 숫자로 주어졌기 때문이다! 그리고 반복될 때마다 증가하려면 i++를 해준다. 그리고 별 증가의 반복문을 만들어주는데 별을 저장해줄 저장소를 tree의 변수로 만들어 주었다. 안쪽 for 문이 돌 때마다 tree에서 _이 더해지도록 <code>tree = tree + " _ ";</code> 을 해주었다.

```
//⭐️을 찍어주는 반복문
for (let j = 1; j <= i * 2 - 1; j++) {
    tree = tree + "*";
  }
```

이 반복문의 구조를 살펴보면 j가 1일 때 1개가 찍히고 i가 2로 증가될 때는 <code>j <= i _ 2 - 1</code> 니까 <code>i _ 2</code>개씩에서 별 1개를 빼준다고 보면된다. 그럼 우리가 원하는 별이 1, 3, 5, 7 이런 조건으로 찍히게 된다. 그리고 이 별들은 가운데 정렬이 되어있으니 시작점에 공백을 넣어주어야 한다.

```
//공백을 찍어주는 반복문
for (let k = 1; k <= level - i; k++) {
    tree = tree + " ";
  }
```

공백을 넣어줄 때는 별이 그려지기 전에 공백이 오기 때문에 공백을 생성하는 for 문은 별 조건문 앞쪽에 선언해주면 된다. k가 길어지는 기준도 level 기준으로 잡아서 똑같이 하면 된다.

### 📃 문제 12번 : 게임 캐릭터 클래스 만들기

- 다음 소스코드에서 클래스를 작성하여 게임 캐릭터의 능력치와 '파이어볼'이 출력되게 만드시오.
  주어진 소스 코드를 수정해선 안됩니다.

```
데이터
<여기에 class를 작성하세요.>

const x = new Wizard(545, 210, 10);
console.log(x.health, x.mana, x.armor);
x.attack();

출력
545 210 10
파이어볼
```

<br>

### 👉 문제 결과

```
// 클래스 작성문
class Wizard {
  constructor(health, mana, armor) {
    this.health = health;
    this.mana = mana;
    this.armor = armor;
  }
  attack() {
    console.log('파이어볼');
  }
}

const x = new Wizard(545, 210, 10);

console.log(x.health, x.mana, x.armor);

x.attack();

// 출력시
545 210 10
파이어볼
```

위자드라는 클래스를 생성할 때 <code>constructor</code>라는 함수를 사용했다.<br>
<code>constructor</code> : 객체에서 인스턴스가 생성이 될 때 반드시 실행이 되는 함수인데 **객체를 초기화 해주는 역할**을 한다.

```
constructor(👉health, mana, armor👈) {
    this.health = health;
    this.mana = mana;
    this.armor = armor;
  }
```

멤버 변수를 선언할 때 <code>this</code>를 사용하는 이유를 간단히 설명하자면, this를 사용하지 않았을 시 constructor의 안에 있는 객체들이 매개변수 안에 있는 것들인지 무엇인지 알 수가 없다. 좀 더 명확하게 해주기 위해 위자드 클래스로 만들어지는 인스턴스는 매개변수에 있는 객체라고 명확하게 해주기 위해서 this 키워드를 사용한 것이다.

```
    this.health = health;
    this.mana = mana;
    this.armor = armor;
```

this를 넣어서 이들이 <code>constructor(health, mana, armor)</code>에 담긴 매개변수라는 것을 명확하게 표현했다.

```
attack() {
    console.log('파이어볼');
  }
```

이것도 '파이어볼'이라는 텍스트가 들어간 인스턴스라고 보면 된다. <br>
이렇게 하면 출력 할 때,

```
const x = new Wizard(545, 210, 10);
console.log(x.health, x.mana, x.armor);
x.attack();
```

인스턴스의 값들 <code>(health=545, mana=210, armor=10)</code>으로 인식하는 것이다. <br> 그리하여 <code>console.log(x.health, x.mana, x.armor)</code>은 x의 멤버변수를 뜻하는 것이다.

### 📃 문제 14번 : 3의 배수 인가요?

- 영희는 친구와 게임을 하고 있습니다. 서로 돌아가며 랜덤으로 숫자를 하나 말하고 그게 3의 배수이면 박수를 치고 아니면 그 숫자를 그대로 말하는 게임입니다. 입력으로 랜덤한 숫자 n이 주어집니다. 만약 그 수가 **3의 배수라면 '짝👏'이라는 글자를, 3의 배수가 아니라면 n을 그대로 출력해 주세요**

```
입출력

입력:3
출력:짝

입력:2
출력:2
```

<br >

### 👉 문제 결과

```
const n = prompt();

if (n % 3 === 0 && n !== 0) {
  console.log('짝');
} else {
  console.log(n);
}
```

이럴 때는 조건문 if를 사용해야 합니다. prompt라는 메서드를 사용하여 입력값을 받아내야 하는데, **n이 3의 배수라는 것을 알게하려면 3으로 나눌 때 나머지값이 0으로 딱 떨어져야 한다.** 그래서 <code>n % 3 === 0</code> 라는 수식을 사용했다. 숫자 0도 3의 배수로 나머지값이 0으로 떨어진 것으로 인식할 수도 있으니까 && 연산자를 사용하여 조건문을 동시에 입력해준다. <code>if (n % 3 === 0 && n !== 0)</code> 그리고 이와 성립하지 않는다면 else 문을 사용하여 그냥 n값 고대로 콘솔창에 찍히도록 하면 된다.

### 📃 문제 16번 : 로꾸꺼

- 문장이 입력되면 거꾸로 출력하는 프로그램을 만들어 봅시다.

```
입출력

입력 : 거꾸로
출력 : 로꾸거
```

<br>

### 👉 문제 결과

```
var str = '거꾸로';

console.log(str.split('').reverse().join(''));
```

`split` : **문자열을 배열로 반환해주는 함수**이다. `split(',')` 이렇게 쉼표를 집어넣으면 쉼표 위주로 문자를 쪼개준다. 만약 '-' 기호를 넣으면 이 기호 위주로 문자열을 배열로 쪼개준다. 문제 결과에서는 split('') 이런 식으로 아무값을 넣어주지 않았으니 거/꾸/로 이렇게 문자열을 쪼개줄 것이다. <br>

`reverse` : **문자열을 반전 상태로 반환해주는 함수**다. split으로 쪼개진 문자열을 거꾸로 반전 상태로 반환해준다. 그래서 '거꾸로'를 '로꾸꺼'로 반환해준다. <br>

`join` : **배열을 문자열로 바꿔주는 함수**이다. split은 join의 반대로 문자열을 배열로 반환해주는 함수지만 join은 배열을 문자열로 바꿔줌으로써 쪼개져서 반전 된 값들을 join을 사용하여 하나의 문자열로 만든다. join 함수에도 매개변수를 집어넣을 수 있는데 split이랑 쓰임이 비슷하다. join함수 괄호안에 `join(',')` 이처럼 쉼표를 넣으면 문자 중간마다 , 쉼표가 찍혀 문자열로 반환해준다. <br>

이렇게 메서드를 연결해서 처리하는 방식을 `메서드체인`이라고 한다.
**⭐메서드체인은 자바스크립트만의 특징**이라고 할 수 있다.
